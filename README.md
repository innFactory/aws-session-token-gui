## Install & Packaging
Clone project:
```
$ git clone https://github.com/innFactory/aws-session-token-gui.git
```

Install dependencies:
```
$ cd aws-session-token-gui
$ npm install
```

Package the app for your local platform:
```
$ npm run package
```
